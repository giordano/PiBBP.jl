# PiBBP — Module.

A toy Julia implementation of Bailey-Borwein-Plouffe (BBP) alogithm for computation of hexadecimal digits of Pi.

# PiBBP.digpi — Function.

Compute $d+1$th hexadecimal digit of $\pi$.

# PiBBP.frac_part — Function.

Fractional part of a real number $x$.

# PiBBP.sigma — Function.

For integer $d$ and $j$ computes the fractional part of [ \sum_{k=0}^d \frac{16^{d-k} \bmod 8k+j}{8k+j} + \sum_{k=d+1}^\infty \frac{1}{16^{k-d} (8k+j)}. ]

# PiBBP.ratio — Function.

For integer $d$, $k$, and $j$ compute [ ( 16^{d-k} \bmod 8k + j ) / (8k + j). ]
